# Dados dos Atletas

Aplicação que recebe informações de um atleta, calcula parâmetros e os exibe para o usuario.

## Descrição

A aplicação rebece dados gerais dos atletas, como:  nome, idade, peso, altura e suas notas. Com base nessas informações atribui ao usuário uma categoria, calcula seu IMC (Índice de Massa Corporal) e a média válida de suas notas, onde o cálculo desconsidera a menor e maior nota. E por fim entrega ao usuário todas as informações agrupadas.

## Funcionalidade

O projeto cria a classe Atleta, coleta os dados da instância atleta e os utiliza nos métodos de instância da classe. Por fim mostra todos os dados do atleta utilizando o console.log().

## Como executar

1. Clone o repositorio `https://github.com/josefelipeas/dados-atletas.git`
2. Instale o ambiente de execução [node](https://nodejs.org/pt/download).
3. Execute o seguinte comando no terminal, no diretório correspondente ao projeto:
```bash
node dados-atletas.js
```
